Contoh Penggunaan Web

Berikut adalah beberapa contoh penggunaan teknologi web seperti CSS, JavaScript, Python, PHP, C++, dan C#.

1. HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <p>Ini adalah contoh halaman HTML sederhana.</p>
    <a href="https://www.example.com">Kunjungi Example</a>
    <img src="gambar.jpg" alt="Contoh Gambar">
</body>
</html>

4. Python (Server-Side)

Python biasanya digunakan di sisi server, berikut adalah contoh sederhana dengan Flask:

# app.py (Flask)
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Python Flask!"

if __name__ == "__main__":
    app.run()

6. Java (Server-Side)

Java juga bisa digunakan untuk membuat aplikasi web, berikut contoh servlet sederhana:

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class HelloWorldServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello, World from Java Servlet!</h1>");
    }
}

7. C++ (Ilustrasi Konsep)

C++ biasanya digunakan untuk pemrograman sistem atau aplikasi desktop. Berikut contoh sederhana C++ yang tidak berjalan di web browser:

// contoh.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World from C++!" << endl;
    return 0;
}

8. C# (ASP.NET Core)

C# digunakan untuk membuat aplikasi web dengan framework ASP.NET Core. Berikut adalah contoh controller sederhana di C#:

// HomeController.cs
using Microsoft.AspNetCore.Mvc;

namespace WebApp.Controllers
{
    public class HomeController : Controller
    {
        public IActionResult Index()
        {
            return View();
        }

        public IActionResult About()
        {
            ViewData["Message"] = "Halaman tentang aplikasi ini.";
            return View();
        }
    }
}

Untuk menjalankan aplikasi ini, pastikan Anda memiliki lingkungan ASP.NET Core dan .NET SDK yang sesuai.

9. Ruby (Sinatra)

Ruby juga bisa digunakan untuk membuat aplikasi web, salah satunya dengan menggunakan framework Sinatra. Berikut contoh sederhana:

# app.rb
require 'sinatra'

get '/' do
  "Hello, Ruby Sinatra!"
end

get '/about' do
  "Ini adalah halaman tentang aplikasi ini."
end

Untuk menjalankan aplikasi ini, buka terminal Anda dan jalankan perintah berikut:

gem install sinatra
ruby app.rb

10. Golang (Server-Side)

Golang juga bisa digunakan untuk membuat server web sederhana. Berikut adalah contoh penggunaannya:

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World from Golang!")
}

func main() {
    http.HandleFunc("/", handler)
    fmt.Println("Server berjalan di http://localhost:8080")
    http.ListenAndServe(":8080", nil)
}

Jalankan script ini, lalu buka browser dan kunjungi http://localhost:8080 untuk melihat hasilnya.
