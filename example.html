Contoh Penggunaan Web

Berikut adalah beberapa contoh penggunaan teknologi web seperti HTML, CSS, JavaScript, Python, PHP, Java, C++, C#, dan Go (Golang).

1. HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang</h1>
    <p>Ini adalah contoh halaman HTML sederhana.</p>
    <a href="https://www.example.com">Kunjungi Example</a>
    <img src="gambar.jpg" alt="Contoh Gambar">
</body>
</html>

Output Contoh HTML:

Selamat Datang

Ini adalah contoh halaman HTML sederhana.

Kunjungi Example

2. CSS (Cascading Style Sheets)

CSS digunakan untuk menata elemen-elemen HTML.

/* Contoh penggunaan CSS */
body {
    background-color: lightblue;
}
h1 {
    color: navy;
    font-size: 2em;
}
p {
    color: darkgray;
}

3. JavaScript

JavaScript digunakan untuk membuat halaman web interaktif.

Ini adalah teks yang akan berubah.

4. Python (Server-Side)

Python biasanya digunakan di sisi server, berikut adalah contoh sederhana dengan Flask:

# app.py (Flask)
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Python Flask!"

if __name__ == "__main__":
    app.run()

5. PHP (Server-Side)

PHP digunakan untuk menangani logika di sisi server.

Berikut adalah contoh script PHP sederhana:

<?php
    echo "Hello, World from PHP!";
?>

6. Java (Server-Side)

Java juga bisa digunakan untuk membuat aplikasi web, berikut contoh servlet sederhana:

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class HelloWorldServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello, World from Java Servlet!</h1>");
    }
}

7. C++ (Ilustrasi Konsep)

C++ biasanya digunakan untuk pemrograman sistem atau aplikasi desktop. Berikut contoh sederhana C++:

// contoh.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World from C++!" << endl;
    return 0;
}

8. C# (ASP.NET Core)

C# digunakan untuk membuat aplikasi web dengan framework ASP.NET Core. Berikut adalah contoh controller sederhana di C#:

// HomeController.cs
using Microsoft.AspNetCore.Mvc;

namespace WebApp.Controllers
{
    public class HomeController : Controller
    {
        public IActionResult Index()
        {
            return View();
        }

        public IActionResult About()
        {
            ViewData["Message"] = "Halaman tentang aplikasi ini.";
            return View();
        }
    }
}

9. Go (Golang)

Go digunakan untuk membangun aplikasi web dengan performa tinggi. Berikut contoh sederhana menggunakan package net/http:

package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World from Golang!")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}

Untuk menjalankan contoh ini, jalankan perintah go run main.go dan akses http://localhost:8080 di browser Anda.
