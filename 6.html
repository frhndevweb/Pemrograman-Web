Materi tentang Java

Pengenalan Java

Java adalah bahasa pemrograman yang berorientasi objek, dikembangkan oleh Sun Microsystems pada tahun 1995. Java dirancang untuk memiliki sedikit ketergantungan pada platform, sehingga dapat dijalankan di berbagai sistem operasi.

Sejarah Singkat Java

Java pertama kali diperkenalkan sebagai bagian dari proyek Green oleh James Gosling dan timnya. Dengan tujuan untuk membuat bahasa yang dapat digunakan di berbagai perangkat, Java telah berkembang pesat dan menjadi salah satu bahasa pemrograman yang paling populer di dunia.

Struktur Dasar Program Java

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Elemen Penting dalam Java

  • Class: Struktur dasar dalam Java
  • Main method: Titik awal eksekusi program
  • Variables: Menyimpan data
  • Data Types: Jenis data (int, double, String, dll.)
  • Control Statements: Struktur kontrol (if, switch, loop)
  • Exception Handling: Mengelola kesalahan dalam program

Contoh Penggunaan Variabel dan Loop

int age = 17;
String name = "Hannaaffiii";

Loop:

for (int i = 0; i < 5; i++) {
    System.out.println("Iterasi ke-" + i);
