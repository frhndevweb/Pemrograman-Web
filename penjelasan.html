Penjelasan Kode Pemrograman Web

1. Front-End (Client-Side)

Kode yang berjalan di browser dan bertanggung jawab untuk tampilan serta interaksi pengguna:

  • HTML: struktur halaman web.
  • CSS: styling dan desain.
  • JavaScript: interaktivitas dan animasi.
  • Framework/Library: React.js, Angular, Vue.js, dll.

2. Back-End (Server-Side)

Kode yang berjalan di server, mengatur logika, database, dan pengolahan data:

  • PHP: membangun website dinamis.
  • Node.js: JavaScript untuk server.
  • Python: framework seperti Django, Flask.
  • Ruby: framework seperti Ruby on Rails.

3. Database

Kode untuk mengelola dan mengakses database:

  • Relasional: MySQL, PostgreSQL.
  • Non-Relasional: MongoDB, Firebase Firestore.
  • Graph Database: Neo4j.

4. Full-Stack Tools

Kombinasi teknologi front-end dan back-end, seperti:

  • MEAN (MongoDB, Express.js, Angular, Node.js).
  • MERN (MongoDB, Express.js, React.js, Node.js).
