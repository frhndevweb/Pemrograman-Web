Materi tentang C++

Pengenalan C++

C++ adalah bahasa pemrograman yang dikembangkan oleh Bjarne Stroustrup pada tahun 1985. C++ adalah bahasa yang berorientasi objek dan merupakan pengembangan dari bahasa C, yang menambahkan fitur-fitur baru seperti kelas dan objek.

Sejarah Singkat C++

C++ awalnya dikembangkan sebagai ekstensi dari C untuk mendukung pemrograman berorientasi objek. Seiring berjalannya waktu, C++ telah menjadi salah satu bahasa pemrograman yang paling banyak digunakan di berbagai bidang, termasuk pengembangan perangkat lunak, game, dan aplikasi sistem.

Struktur Dasar Program C++

#include <iostream>

using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Elemen Penting dalam C++

  • Header Files: File yang menyertakan fungsi standar
  • Main Function: Titik awal eksekusi program
  • Variables: Menyimpan data
  • Data Types: Jenis data (int, float, char, dll.)
  • Control Statements: Struktur kontrol (if, switch, loop)
  • Classes: Mendefinisikan tipe data baru dan metode terkait

Contoh Penggunaan Variabel dan Loop

Variabel:

int age = 17;
string name = "Hannaaffiii";

Loop:

for (int i = 0; i < 5; i++) {
    cout << "Iterasi ke-" << i << endl;
}
