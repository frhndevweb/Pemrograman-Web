Struktur Dasar Python

Struktur Kontrol

if age >= 17:
    print("Dewasa")
else:

Looping

for i in range(5):
    print(i)
        
while age < 17:
    print(age)
    age += 1

List, Tuple, dan Dictionary

# List

# Tuple

# Dictionary

Modul dan Pustaka Eksternal

Anda dapat mengimpor modul untuk menggunakan fungsionalitas tambahan:

Untuk menginstal pustaka eksternal, gunakan pip:

pip install nama_pustaka

Pemrograman Berorientasi Objek (OOP)

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, my name is {self.name}."

hanna = Person("Hannaaffiii", 17)
print(hanna.greet())

Penanganan Kesalahan

Gunakan try dan except untuk menangani kesalahan:

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Tidak bisa membagi dengan nol!")
