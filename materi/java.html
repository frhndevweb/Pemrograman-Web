Materi tentang Java

Pengenalan Java

Java adalah bahasa pemrograman yang berorientasi objek, dikembangkan oleh Sun Microsystems pada tahun 1995. Java dirancang untuk memiliki sedikit ketergantungan pada platform, sehingga dapat dijalankan di berbagai sistem operasi.

Sejarah Singkat Java

Java pertama kali diperkenalkan sebagai bagian dari proyek Green oleh James Gosling dan timnya. Dengan tujuan untuk membuat bahasa yang dapat digunakan di berbagai perangkat, Java telah berkembang pesat dan menjadi salah satu bahasa pemrograman yang paling populer di dunia.

Struktur Dasar Program Java

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Elemen Penting dalam Java

  • Class: Struktur dasar dalam Java
  • Main method: Titik awal eksekusi program
  • Variables: Menyimpan data
  • Data Types: Jenis data (int, double, String, dll.)
  • Control Statements: Struktur kontrol (if, switch, loop)
  • Exception Handling: Mengelola kesalahan dalam program

Contoh Penggunaan Variabel dan Loop

Variabel:

int age = 17;
String name = "Hannaaffiii";

Loop:

for (int i = 0; i < 5; i++) {
    System.out.println("Iterasi ke-" + i);
}

Struktur Kontrol

Contoh If-Else:

if (age >= 17) {
    System.out.println("Dewasa");
} else {
    System.out.println("Anak-anak");
}

Contoh Switch:

switch (age) {
    case 17:
        System.out.println("Usia 17 tahun");
        break;
    default:
        System.out.println("Usia tidak terdefinisi");
}

Metode dalam Java

public static void greet(String name) {
    System.out.println("Hello, " + name + "!");
}

// Memanggil metode
greet("Hannaaffiii");

Penanganan Exception

try {
    int[] numbers = {1, 2, 3};
    System.out.println(numbers[5]); // Ini akan menimbulkan ArrayIndexOutOfBoundsException
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Kesalahan: " + e.getMessage());
}

Koleksi dalam Java

import java.util.ArrayList;

ArrayList fruits = new ArrayList<>();
fruits.add("apel");
fruits.add("jeruk");
fruits.add("pisang");

for (String fruit : fruits) {
    System.out.println(fruit);
}

Pemrograman Berorientasi Objek (OOP)

class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    void display() {
        System.out.println("Nama: " + name + ", Umur: " + age);
    }
}

Person hanna = new Person("Hannaaffiii", 17);
hanna.display();
