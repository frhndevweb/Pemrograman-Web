Materi tentang HTML

Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Dengan HTML, kita dapat menentukan elemen seperti teks, gambar, video, tautan, hingga formulir pada sebuah website.

Sejarah Singkat HTML

HTML pertama kali dikembangkan oleh Tim Berners-Lee pada tahun 1991 sebagai cara untuk mengatur dan menghubungkan dokumen di web. HTML versi awal sangat sederhana dan hanya mendukung elemen dasar seperti teks dan tautan. Kini, dengan HTML5, pengembang dapat menggunakan elemen-elemen modern untuk mendukung multimedia, grafis, dan fitur lainnya yang lebih kompleks.

Struktur Dasar HTML

Setiap dokumen HTML memiliki struktur dasar seperti berikut:

<!DOCTYPE html> 
<html lang="id"> 
<head> 
    <title>Judul Halaman</title> 
</head>
<body> 
    <h1>Hello World</h1>
</body>
</html>

Setiap elemen HTML biasanya terdiri dari tag pembuka, konten, dan tag penutup.

Elemen Penting

Berikut adalah elemen-elemen HTML yang sering digunakan:

  • <h1> hingga <h6>: Untuk menampilkan judul dengan tingkat hierarki tertentu.
  • <p>: Untuk membuat paragraf teks.
  • <a>: Untuk membuat tautan atau link ke halaman lain.
  • <img>: Untuk menampilkan gambar.
  • <ul> dan <ol>: Untuk membuat daftar (unordered list dan ordered list).
  • <table>: Untuk membuat tabel.

Contoh Penggunaan Daftar dan Tabel

Daftar Tidak Terurut:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

Tabel:

<table>
    <tr>
        <th>Nama</th>
        <th>Umur</th>
    </tr>
    <tr>
        <td>Hannaaffiii</td>
        <td>17</td>
    </tr>
</table>

Atribut HTML

Atribut memberikan informasi tambahan pada elemen HTML. Contohnya:

<a href="https://www.example.com">Kunjungi Situs</a>
<img src="gambar.jpg" alt="Deskripsi Gambar">

Atribut href digunakan pada elemen <a> untuk menentukan tujuan link, sedangkan atribut src pada <img> menentukan lokasi file gambar.

Elemen Multimedia

HTML5 mendukung elemen untuk menyematkan multimedia seperti video dan audio.

Contoh Video:

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Browser Anda tidak mendukung video tag.
</video>

Contoh Audio:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Browser Anda tidak mendukung audio tag.
</audio>

Formulir

Formulir digunakan untuk mengumpulkan input dari pengguna:

<form action="/submit" method="post">
    <label for="nama">Nama:</label>
    <input type="text" id="nama" name="nama">
    <input type="submit" value="Kirim">
</form>
