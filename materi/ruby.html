Materi tentang Ruby

Pengenalan Ruby

Ruby adalah bahasa pemrograman dinamis yang terkenal karena kesederhanaan dan produktivitasnya. Ruby dirancang untuk menjadi mudah dipahami dan digunakan, dengan sintaks yang bersih dan elegan.

Sejarah Singkat Ruby

Ruby pertama kali dikembangkan oleh Yukihiro Matsumoto (Matz) di Jepang pada tahun 1995. Tujuannya adalah untuk menciptakan bahasa pemrograman yang menggabungkan kelebihan bahasa pemrograman seperti Perl, Smalltalk, dan Ada. Sejak saat itu, Ruby telah berkembang pesat dan menjadi salah satu bahasa yang populer, terutama dalam pengembangan web dengan framework Ruby on Rails.

Struktur Dasar Program Ruby

Program Ruby yang paling sederhana hanya terdiri dari satu baris kode. Berikut ini adalah contoh program Ruby yang mencetak "Hello, World":

puts "Hello, World"

Penjelasan:

  • puts: Metode yang digunakan untuk mencetak output ke konsol, diakhiri dengan newline.

Fitur Utama Ruby

Berikut ini adalah beberapa fitur utama Ruby yang membuatnya unik:

  • Objek Sepenuhnya: Semuanya adalah objek di Ruby, termasuk angka dan nil.
  • Blok dan Procs: Menyediakan cara untuk menyimpan dan meneruskan kode sebagai parameter.
  • Metaprogramming: Kemampuan untuk memanipulasi program saat runtime, membuat kode lebih fleksibel.
  • Sintaks yang Elegan: Sintaks Ruby dirancang untuk kemudahan pembacaan dan penulisan.

Penggunaan Thread

Ruby mendukung threading, yang memungkinkan eksekusi beberapa bagian kode secara bersamaan. Berikut adalah contoh sederhana penggunaan thread:

thread = Thread.new do
    puts "Hello from thread"
end

thread.join # Menunggu thread selesai
puts "Main thread finished"

Dalam contoh ini, kita membuat thread baru dan menjalankan kode di dalam blok. Metode join digunakan untuk menunggu hingga thread selesai eksekusi.

Penggunaan Blok dan Procs

Ruby menyediakan fitur blok yang memungkinkan Anda mengirimkan sepotong kode sebagai parameter ke metode. Contoh penggunaan blok:

def greet
    yield "Hello from block"
end

greet { |message| puts message }

Dalam contoh ini, metode greet menerima blok dan memanggilnya dengan yield.

Metaprogramming di Ruby

Ruby mendukung metaprogramming, yang memungkinkan Anda untuk memodifikasi kelas dan objek saat runtime. Contoh metaprogramming sederhana:

class DynamicClass
    define_method(:dynamic_method) do |name|
        "Hello, #{name}!"
    end
end

obj = DynamicClass.new
puts obj.dynamic_method("Ruby")

Dalam contoh ini, kita menggunakan define_method untuk mendefinisikan metode secara dinamis.

Contoh Program Lengkap Ruby

Berikut adalah contoh program Ruby yang lebih kompleks yang menggabungkan berbagai konsep di atas:

class Counter
    def initialize
        @count = 0
    end

    def increment
        @count += 1
    end

    def display
        puts "Current count: #{@count}"
    end
end

counter = Counter.new

threads = []
5.times do
    threads << Thread.new do
        counter.increment
        counter.display
    end
end

threads.each(&:join) # Menunggu semua thread selesai
puts "All threads finished"

Program ini mendefinisikan kelas Counter yang memiliki metode untuk mengincrement dan menampilkan nilai. Lima thread dibuat untuk meningkatkan nilai counter dan menampilkannya.
