Materi tentang C#

Pengenalan C#

C# adalah bahasa pemrograman yang dikembangkan oleh Microsoft pada tahun 2000. C# merupakan bahasa pemrograman berorientasi objek yang digunakan untuk berbagai aplikasi, termasuk pengembangan aplikasi desktop, aplikasi web, game, dan mobile. Situs lengkap tentang C#

Sejarah Singkat C#

C# pertama kali diperkenalkan sebagai bagian dari .NET Framework dan telah berkembang menjadi salah satu bahasa utama untuk pengembangan di platform Microsoft. C# dirancang untuk menjadi sederhana, modern, dan memiliki sintaks yang mirip dengan bahasa C++ dan Java.

Struktur Dasar Program C#

using System;

class Program {
    static void Main(string[] args) {
        Console.WriteLine("Hello, World!");
    }
}

Elemen Penting dalam C#

  • Namespaces: Mengatur dan mengelompokkan kelas dan fungsi
  • Main Function: Titik awal eksekusi program
  • Variables: Menyimpan data dengan tipe tertentu
  • Data Types: Jenis data seperti int, float, string, dll.
  • Control Statements: Struktur kontrol (if, switch, loop)
  • Classes: Mendefinisikan objek dan metode terkait

Contoh Penggunaan Variabel dan Loop di C#

Variabel:

int age = 17;
string name = "Hannaaffiii";

Loop:

for (int i = 0; i < 5; i++) {
    Console.WriteLine("Iterasi ke-" + i);
}

Konsep Tambahan di C#

1. Properti

Properti adalah cara untuk mengakses data dalam kelas dengan aman. C# mendukung properti otomatis untuk mempermudah pengelolaan data.

class Person {
    public string Name { get; set; }
    public int Age { get; set; }
}

Person person = new Person();
person.Name = "Hannaaffiii";
person.Age = 17;

Console.WriteLine(person.Name);

2. Exception Handling

Exception handling di C# menggunakan try-catch untuk menangkap dan menangani kesalahan dalam program.

try {
    int number = int.Parse("abc");  // Ini akan menimbulkan kesalahan
} catch (FormatException e) {
    Console.WriteLine("Kesalahan format: " + e.Message);
}

3. LINQ (Language Integrated Query)

LINQ adalah fitur C# yang memungkinkan kita melakukan query pada koleksi data seperti array, list, atau database dengan cara yang lebih mudah.

int[] numbers = { 1, 2, 3, 4, 5 };
var evenNumbers = from num in numbers
                  where num % 2 == 0
                  select num;

foreach (var num in evenNumbers) {
    Console.WriteLine(num);
}

4. Struktur Kontrol

Contoh If-Else:
if (age >= 17) {
    Console.WriteLine("Dewasa");
} else {
    Console.WriteLine("Anak-anak");
}
Contoh Switch:
switch (age) {
    case 17:
        Console.WriteLine("Usia 17 tahun");
        break;
    default:
        Console.WriteLine("Usia tidak terdefinisi");
        break;
}

5. Fungsi

static void Greet(string name) {
    Console.WriteLine("Hello, " + name + "!");
}

// Memanggil fungsi
Greet("Hannaaffiii");

6. Pemrograman Berorientasi Objek (OOP)

class Animal {
    public string Name { get; set; }

    public void Speak() {
        Console.WriteLine(Name + " berkata: Meow!");
    }
}

Animal cat = new Animal();
cat.Name = "Kucing";
cat.Speak();
