Materi tentang PHP

Pengenalan PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang sering digunakan untuk pengembangan web.

Instalasi PHP

Anda dapat mengunduh PHP dari situs resmi PHP. Untuk pengembangan lokal, Anda dapat menggunakan XAMPP atau WAMP.

Struktur Dasar PHP

<?php
echo "Hello, World!";
?>

Variabel dan Tipe Data

$nama = "hannaaffiii";  // String
$umur = 17;      // Integer
$tinggi = 1.75;  // Float
$mahasiswa = true; // Boolean

Fungsi

function greet($name) {
    return "Hello, $name!";
}

echo greet("Hannaaffiii");

Struktur Kontrol

if ($umur >= 17) {
    echo "Dewasa";
} else {
    echo "Anak-anak";
}

Looping

for ($i = 0; $i < 5; $i++) {
    echo $i;
}

while ($umur < 30) {
    echo $umur;
    $umur++;
}

Array dan Associative Array

$array = array("apel", "jeruk", "pisang");

$associativeArray = array(
    "nama" => "Hannaaffiii",
    "umur" => 17
);
echo $associativeArray["nama"];

Superglobals

echo $_SERVER['PHP_SELF']; // Menampilkan nama file saat ini
echo $_GET['nama']; // Mengakses data dari URL
echo $_POST['nama']; // Mengakses data dari formulir

Penanganan Formulir

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nama = $_POST['nama'];
    echo "Nama Anda adalah: $nama";
}

Contoh formulir:

<form method="POST" action="process.php">
    <input type="text" name="nama">
    <input type="submit" value="Kirim">
</form>

Koneksi Database dengan PDO

try {
    $pdo = new PDO('mysql:host=localhost;dbname=database_name', 'username', 'password');
    $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
    $stmt = $pdo->query('SELECT * FROM users');

    while ($row = $stmt->fetch()) {
        echo $row['name'];
    }
} catch (PDOException $e) {
    echo 'Connection failed: ' . $e->getMessage();
}

Penanganan Kesalahan

set_error_handler(function ($errno, $errstr) {
    echo "Error: [$errno] $errstr";
});
